# User Flows (MVP)

## 1. Первый вход

1. Пользователь открывает приложение.
2. Видит кнопку **Login with Google** (AuthGate).
3. После OAuth редиректа попадает на главную страницу.

## 2. Создание цели

1. На главной нажимает **Add goal**.
2. Заполняет:
   - название;
   - тип цели (time/counter/check);
   - период;
   - направление (gte/lte);
   - даты начала/окончания;
   - контекст (обязателен);
   - теги (опционально).
3. Цель сохраняется и открывается экран цели.

## 3. Логирование с главного экрана

- **Counter**: ввод числа + кнопка `+` (если поле пустое — добавляет 1).
- **Time**: кнопки play/stop запускают/останавливают таймер.
- **Check**: кнопка ✓ создаёт событие состояния.

Фильтры доступны по контекстам, тегам и статусу.

## 4. Глобальный таймер

1. Нажать **Play** в нижней панели.
2. Указать/создать контекст и (опционально) теги.
3. Запустить таймер без привязки к цели.

## 5. Экран цели

- Просмотр прогресса по текущему периоду.
- Редактирование названия и даты окончания.
- Управление тегами.
- Архивирование цели.
- Ручное добавление событий и удаление существующих.

## 6. Экран статистики

- В текущей версии отображается заглушка.
