# User flows (MVP)

## Первый вход
1. Пользователь открывает приложение.
2. Входит через Google.
3. Попадает на главный экран.

Если целей нет — видит пустое состояние и кнопку создания цели.

## Создание цели
1. На главном экране нажать `Add goal`.
2. Заполнить форму: название, тип, период, цель, даты, контекст.
3. (Опционально) добавить теги.
4. Сохранить — цель появляется в списке.

## Логирование на главной
- Counter: ввести число и нажать `+` (по умолчанию 1).
- Time: `Play`/`Stop` на карточке цели.
- Check: переключатель Done/Not done.

## Глобальный таймер
1. Нажать `Play` в нижней панели.
2. Выбрать/создать контекст.
3. (Опционально) добавить теги.
4. Старт/стоп таймера.

## Детали цели
1. Открыть цель.
2. Посмотреть прогресс и события.
3. Добавить событие вручную при необходимости.
