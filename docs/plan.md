# Текущее состояние проекта

## Реализовано

- Google OAuth через Supabase.
- CRUD для целей (создание, редактирование, архивирование).
- Контексты и теги (создаются на лету).
- Главный экран со списком активных целей, фильтрами и быстрым логированием.
- Глобальный таймер для логирования времени без цели.
- Экран цели с прогрессом и событиями.
- Базовый пересчёт `goal_periods` на клиенте.

## Известные ограничения

- Экран статистики (`/stats`) пока в формате заглушки.
- Нет отдельного экрана управления контекстами и тегами.

## Точки входа

- `/` — главная
- `/goals/new` — создание цели
- `/goals/[id]` — детали цели
- `/stats` — статистика (заглушка)
