# Текущее состояние

## Реализовано
- Google OAuth login/logout.
- Цели: создание (time/counter/check), привязка тегов, контекст на лету.
- Главный экран: список активных целей, быстрые действия, фильтры.
- Детали цели: прогресс, недавние события, ручной ввод, редактирование тегов.
- Глобальный таймер без цели (контекст обязателен, теги опциональны).
- Пересчет goal periods для статусов.
- Тесты: Vitest (unit/integration) + Playwright (e2e).
- CI: GitHub Actions workflow на master.
- Статистика: фильтры и графики.
- Сортировка целей по статусу на главном экране.
