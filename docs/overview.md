# Обзор проекта

Habimori — минималистичный трекер целей и времени. Все данные принадлежат пользователю и защищены RLS в Supabase.

## Основные возможности

- Google OAuth (Supabase): вход/выход, автоматический upsert профиля пользователя.
- Цели трёх типов:
  - `time` (минуты)
  - `counter` (количество)
  - `check` (факт выполнения)
- Периоды целей: день / неделя / месяц.
- Контекст обязателен, теги — опциональны (создаются на лету).
- Главный экран:
  - список активных целей, текущий прогресс и статус по текущему периоду;
  - фильтры по контексту, тегам и статусу;
  - быстрые действия: +счётчик, play/stop таймера, чекбокс.
- Глобальный таймер (нижняя панель) для логирования времени без цели.
- Экран цели:
  - редактирование названия и даты окончания;
  - управление тегами;
  - архивирование;
  - ручное добавление событий (time/counter/check) и удаление событий.
- Экран статистики (`/stats`) пока содержит заглушку.

## Основные маршруты

- `/` — список активных целей и быстрый логинг.
- `/goals/new` — создание цели.
- `/goals/[id]` — детали цели и события.
- `/stats` — статистика (пока заглушка).
- `/auth/callback` — завершение OAuth.
